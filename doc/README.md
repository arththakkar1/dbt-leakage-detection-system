# Documentation

This directory contains the comprehensive documentation for the DBT Leakage Detection System, structured according to the Diátaxis framework.

## Structure
- **`explanation/`**: Theoretical context, system architecture, model workings, and workflows. Focuses on understanding *why* and *how* the system works.
- **`how-to/`**: Goal-oriented guides that help you solve specific problems or accomplish tasks (e.g., deploying the system).
- **`reference/`**: Technical descriptions and machinery details (e.g., API contracts, data schemas, rules engines).
- **`tutorials/`**: Learning-oriented guides that take you by the hand through the process of setting up and running the project.
